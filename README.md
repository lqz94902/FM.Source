## 如何修改电台在收藏夹中的排序？
 进入CouchDB数据库找到与电台收藏相关的数据内容，找到与电台相关的数据列表，其中m_created_at和m_updated_at分别为每一条收藏记录的创建和修改的时间戳，软件就是根据这个时间戳按照从新到久的顺序进行排序的。也就是所如果我们需要修改某一个电台在软件电台列表中的位置，只需要将该条记录的时间戳修改为所需位置前后两条记录之间即可。
